# 曼荼羅 / Mandara

> 思維路由地圖 · A personal mandala of thinking systems

`mandara` is a living atlas of thinking frameworks — 23 paradigms organised across 6 ranges (Latticework / Curriculum / Cynefin / Loop / Anti-Bias / Container/Frame), with a deeper drill-down per paradigm.

The project takes its name from 曼荼羅 (mandala) — the structural arrangement of cosmos and consciousness used in Buddhist iconography. Like a mandala, this atlas places every element in relation to every other, lets you read by zooming into any single piece, and reveals new connections through the act of arrangement.

## Two work tracks framing

Every paradigm card and every HC drill-down is annotated for two contexts:

- **個人品牌 / Personal brand** — writing, content, positioning, audience
- **APAC AIA NGO governance** — multi-stakeholder decisions, cross-cultural negotiation, systemic change

Filter pills surface the subset relevant to each track + cross-cutting situations (cross-cultural, crisis, reflection).
